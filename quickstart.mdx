# 🚀 Eco Routes SDK Quickstart  

This guide will help you integrate the **Eco Routes SDK** to enable cross-chain stablecoin transfers.  

✅ **Install the SDK**  
✅ **Create a transfer intent**  
✅ **Request and apply a quote**  
✅ **Publish the intent**  

---

## 1️⃣ Install the SDK  

To get started, install the **Eco Routes SDK** and its peer dependency:  

```bash install-sdk.sh
npm install @eco-foundation/routes-sdk @eco-foundation/routes-ts

🔄 Keep Dependencies Updated

Ensure you’re using the latest version of the SDK to stay compatible with the latest contract updates:

npm install @eco-foundation/routes-ts@latest
npm install @eco-foundation/routes-sdk@latest

2️⃣ Create a Transfer Intent

An intent represents the desired transfer outcome on a destination chain.

🔹 Import RoutesService

import { RoutesService } from '@eco-foundation/routes-sdk';

🔹 Define Transfer Parameters

const address = '0xYourEthereumAddress';
const originChainID = 10;  // Optimism
const destinationChainID = 8453;  // Base
const spendingToken = RoutesService.getStableAddress(originChainID, 'USDC');
const receivingToken = RoutesService.getStableAddress(destinationChainID, 'USDC');
const spendingTokenLimit = BigInt(10000000);  // 10 USDC
const amount = BigInt(1000000);  // 1 USDC

🔹 Initialize RoutesService and Create Intent

const routesService = new RoutesService();

const intent = routesService.createSimpleIntent({
  creator: address,
  originChainID,
  spendingToken,
  spendingTokenLimit,
  destinationChainID,
  receivingToken,
  amount,
  recipient: address,  // Optional, defaults to the creator
});

3️⃣ Request and Apply a Quote

To optimize fulfillment, request quotes from solvers.

🔹 Import Quoting Utilities

import { OpenQuotingClient, selectCheapestQuote } from '@eco-foundation/routes-sdk';

🔹 Initialize Quoting Client

const openQuotingClient = new OpenQuotingClient({ dAppID: 'your-dapp-id' });

🔹 Fetch Available Quotes

const quotes = await openQuotingClient.requestQuotesForIntent(intent);
if (!quotes.length) {
  throw new Error('No quotes available');
}

🔹 Select the Best Quote and Apply It

const selectedQuote = selectCheapestQuote(quotes);
const intentWithQuote = routesService.applyQuoteToIntent({ intent, quote: selectedQuote });

📌 You can implement custom logic for selecting a quote based on your criteria.

4️⃣ Publish the Intent

Once the intent is prepared, publish it to the origin chain using your preferred web3 library.

// Example using ethers.js
import { ethers } from 'ethers';

const provider = new ethers.JsonRpcProvider("https://your-rpc-url");
const signer = new ethers.Wallet("your-private-key", provider);

const tx = await signer.sendTransaction({
  to: intentWithQuote.spendingToken,
  value: intentWithQuote.amount,
});

await tx.wait();
console.log("Intent published:", tx.hash);
